What is an IFSC Code?

আইএফএসসি কোড কী?

IFSC (Indian Financial System Code) is an 11-digit alphanumeric code that uniquely identifies a bank branch participating in various online money transfer options in India.

আইএফএসসি (ইন্ডিয়ান ফিনান্সিয়াল সিস্টেম কোড) হল একটি ১১-ডিজিটের আলফানিউমেরিক কোড যা ভারতে বিভিন্ন অনলাইন অর্থ স্থানান্তর বিকল্পে অংশগ্রহণকারী একটি ব্যাংক শাখাকে অনন্যভাবে চিহ্নিত করে।

Structure of IFSC

আইএফএসসির গঠন

The 11-character IFSC is structured as follows:

  • First 4 characters: Bank code (alphabetic)
  • 5th character: 0 (zero, reserved for future use)
  • Last 6 characters: Branch code (alphanumeric)

Uses of IFSC

আইএফএসসি এর ব্যবহার

IFSC is used for:

  • NEFT (National Electronic Funds Transfer)
  • RTGS (Real Time Gross Settlement)
  • IMPS (Immediate Payment Service)
  • UPI (Unified Payments Interface) transactions
